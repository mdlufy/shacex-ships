# SpaceX Ships

Simple single-page application, consists of two pages: ships list page, ship detail page

## Key features

- Elements list page
- Element detail page
- Filter panel
- Pagination
- Caching requests
- State management

## To dev

- `cd spacex-ships`
- `npm i`
- `npm start`
